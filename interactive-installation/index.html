Interactive Installation

不要啦倒｜啦啦隊體驗互動遊戲

站上螢幕前，你就是球場上的啦啦隊員！

跟著提示做出加油姿勢，用你的動作，帶動全場氣氛，體驗成為賽場上的熱血感！

聲色場所｜體感互動設計

以體感辨識作為輸入，將使用者的姿勢與動作映射到音樂與視覺回饋，你用線條與顏色說話，當下的狀態會以聲音回應你。你怎麼下筆、怎麼用色，會帶來不同的聲響，幫你更清楚現在的自己。

海要釣嗎｜釣魚翁互動遊戲

為傳達海洋資源永續利用和判斷什麼魚種能釣的重要性，我們決定透過互動遊戲讓觀眾成為釣魚的人，主動去判別何為不可釣的魚。
